# Bebotph

Monorepo for Bebotph — Flutter mobile app + Next.js web store.

## Structure

```
Bebotph/
├── mobile/       # Flutter app (iOS + Android)
└── webstore/     # Next.js + Shopify Storefront (hosted on Vercel)
```

## Mobile (Flutter)

```bash
cd mobile
flutter run
```

## Web Store (Next.js)

```bash
cd webstore
npm install
npm run dev
```
